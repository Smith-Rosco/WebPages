一、OLED液晶屏

        OLED，即有机发光二极管（Organic Light-Emitting Diode），又称为有机电激光显示（Organic Electroluminesence Display, OELD）。因为具备轻薄、省电等特性，因此从2003 年开始，这种显示设备在 MP3 播放器上得到了广泛应用，而对于同属数码类产品的 DC 与手机，此前只是在一些展会上展示过采用 OLED 屏幕的工程样品。自 2007 年后，寿命得到很大提高，具备了许多 LCD 不可比拟的优势。
        目前市面上最常见的是 0.91 / 0.96 / 1.3 寸液晶串口屏，常用的驱动接口 I2C / SPI 接口。基本都是不带字库的，需要自己生成字库。

二、SH1106 I2C 驱动

1、SH1106 简介

特征：

  • 支持最大 132 X 64 点矩阵面板

  • 嵌入式 132 X 64 位 SRAM

  • 工作电压：
    逻辑电源：VDD1 = 1.65V-3.5V
    DC-DC电源：VDD2 = 3.0V-4.2V
    OLED工作电源：
    外部VPP电源 = 6.4V-14.0V
    内部VPP发生器 = 6.4V-9.0V

  • 最大段输出电流：200mA

  • 最大公共灌电流：27mA

  • 8 位 6800 系列并行接口，8 位 8080 系列并行接口，3 线和 4 线串行外设接口，400KHz 快速 I2C 总线接口

  • 可编程帧频和复用率

  • 行重映射和列重映射（ADC）

  • 垂直滚动

  • 片内振荡器

  • 可编程内部电荷泵电路输出

  • 单色无源OLED面板上的256步对比度控制

  • 低功耗
    睡眠模式：< 5μA
    VDD1 = 0V，VDD2 = 3.0V – 4.2V：< 5μA
    VDD1,2 = 0V，VPP = 6.4V –14.0V：< 5μA

  • 广泛的工作温度范围：-40至+ 85°C

  • 可提供COG形式，厚度：300mm

        SH1106 是具有用于有机 / 聚合物发光二极管点矩阵图形显示系统的控制器的单芯片CMOS OLED / PLED 驱动器。 SH1106 由 132 个段，64 个公共端组成，可支持 132 X 64 的最大显示分辨率。它是为公共阴极型 OLED 面板设计的。
        SH1106 嵌入了对比度控制，显示 RAM 振荡器和高效的 DC-DC 转换器，从而减少了外部组件的数量和功耗。 SH1106 适用于各种紧凑型便携式应用，例如手机的子显示屏，计算器和 MP3 播放器等。

2、I2C 通信接口

        SH1106 可以通过标准 I2C 总线传输数据，并且仅在通信中具有从机模式。 可以将命令或 RAM 数据写入芯片，并且可以从芯片中读取状态和 RAM 数据。

IM0IM1IM2TypeCSA0RDWRD0D1D2 to D7
010I2C 接口拉低SA0--SCLSDA(Hz)

        注意：“-” 引脚必须始终为高电平或低电平。 建议使用 D7 - D2 连接 VDD1 或 VSS。 也可以使 D7 - D2 保持不连接状态。

        SH1106 支持读和写访问。 R / W 位是从机地址的一部分。在 I2C 总线上传输任何数据之前，应首先对应该响应的设备进行寻址。 SH1106 保留了两个 7 位从地址（0111100 和 0111101）。通过将输入 SA0 连接到逻辑 0（VSS）或 1（VDD1），可以设置从机地址的最低有效位。 I2C 总线协议如图 7 所示。该序列从 I2C 总线主机的启动条件（S）开始，后跟从机地址。具有相应地址的所有从机并行确认，所有其他从机将忽略 I2C 总线传输。确认后，将跟随一个或多个命令字，这些命令字定义了被寻址从站的状态。一个命令字由一个定义 Co 和 D / C 的控制字节（注1）以及一个数据字节（见下图）组成。最后一个控制字节被标记为已清除的最高有效位，即连续位 Co。在控制字节被清除后的 Co 位之后，将仅跟随数据字节。 D / C 位的状态定义将数据字节解释为命令还是 RAM 数据。总线上的所有已寻址从站也会确认控制字节和数据字节。在最后一个控制字节之后，根据 D / C 位设置，可能会跟随一系列显示数据字节或命令数据字节。如果 D / C 位设置为1，则这些显示字节将存储在显示 RAM 中数据指针指定的地址处。数据指针将自动更新，并将数据定向到预期的 SH1106 器件。如果最后一个控制字节的 D / C 位设置为“ 0”，则将对这些命令字节进行解码，并根据收到的命令更改设备的设置。每个字节之后的确认仅由被寻址的从站进行。传输结束时， I2C 总线主机发出停止条件（P）。如果从机地址中的 R / W 位置 1，则芯片将根据上次写访问期间发送的 D / C 位在从机地址之后立即输出数据。如果主机在一个字节后未生成任何确认，则驱动程序将停止将数据传输到主机。


注意1：

  1. Co ＝“ 0”：最后一个控制字节，仅跟随数据字节，
    Co ＝“ 1”：接下来的两个字节是数据字节和另一个控制字节；
  2. D / C ＝“ 0”：数据字节用于命令操作，
    D / C ＝“ 1”：数据字节用于RAM操作。

三、SSD1306 I2C 驱动

1、SSD1306 简介

        SSD1306 是具有用于有机/聚合物发光二极管点矩阵图形显示系统的控制器的单芯片 CMOS OLED / PLED 驱动器。 它由 128 个段和 64 个公用部分组成。 该 IC 设计用于普通阴极型 OLED 面板。

        SSD1306 嵌入了对比度控制，显示 RAM 和振荡器，从而减少了外部组件的数量和功耗。 它具有 256 级亮度控制。 数据/命令通过硬件可选的 6800/8000 系列兼容并行接口，I2C 接口或串行外围接口从通用 MCU 发送。 它适用于许多紧凑的便携式应用，例如手机子显示屏，MP3 播放器和计算器等。

特征：

  • 分辨率：128 x 64点矩阵面板
  • 电源
    用于 IC 逻辑的 VDD = 1.65V 至 3.3V
    VCC = 7V 至 15V 用于面板驱动
  • 用于矩阵显示
    OLED驱动输出电压，最大15V
    段最大源电流：100uA
    通用最大灌电流：15mA
    256级对比度亮度电流控制
  • 嵌入式128 x 64位SRAM显示缓冲区
  • 引脚可选的MCU接口：
    8位6800/8080系列并行接口
    3/4线串行外设接口
    I2C接口
  • 屏幕保存的水平和垂直方向连续滚动功能
  • RAM写入同步信号
  • 可编程帧率和多路复用率
  • 行重映射和列重映射
  • 片内振荡器
  • COG和COF的芯片布局
  • 工作温度范围广：-40°C至85°C

2、I2C 通信接口

        SSD1306 MCU 接口包含 8 个数据引脚和 5 个控制引脚。 下表总结了不同接口模式下的引脚分配。 可以通过BS [2：0]引脚上的硬件选择来设置不同的MCU模式（关于BS [2：0]的设置，请参见下表）。

        I2C 通信接口由从机地址位 SA0，I2C 总线数据信号 SDA（用于输出的 SDAOUT / D2 和用于输入的 SDAIN / D1）和 I2C 总线时钟信号 SCL（D0）组成。数据和时钟信号都必须连接到上拉电阻。 RES＃用于设备初始化。

从机地址位（SA0）
        在通过 I2C 总线发送或接收任何信息之前，SSD1306 必须识别从机地址。设备将以以下字节格式响应从机地址，然后是从机地址位（“ SA0”位）和读/写选择位（“ R / W＃”位）

b7b6b5b4b3b2b1b0
011110SA0读/写R / W＃

        “ SA0 ” 位提供从机地址的扩展位。可以选择“ 0111100”或“ 0111101”作为 SSD1306 的从地址。 D / C＃引脚用作从机地址选择的 SA0。 “ R / W＃”位用于确定 I2C 总线接口的操作模式。 R / W＃= 1，它处于读取模式。 R / W＃= 0，处于写入模式。
