Getting started

Quick start

Link pico.css manually or via CDN for a dependency-free setup, or use NPM or Composer for advanced usage.

Content
  • Install manually
  • Usage from CDN
  • Install with NPM
  • Install with Composer
  • Starter HTML template

There are 4 ways to get started with pico.css:

Install manually#

Download Picoand link /css/pico.min.css in the <head>of your website.

<linkrel="stylesheet"href="css/pico.min.css">

Usage from CDN#

Alternatively, you can use jsDelivr CDN to linkpico.min.css.

<linkrel="stylesheet"href="https://cdn.jsdelivr.net/npm/@picocss/pico@2/css/pico.min.css">

Install with NPM#

npminstall @picocss/pico

Or

yarnadd @picocss/pico

Then, import Pico into your SCSS file with@use:

@use"pico";

Learn more about the customization with Sass.

Install with Composer#

composer require picocss/pico

Starter HTML template#

<!doctypehtml><htmllang="en"><head><metacharset="utf-8"><metaname="viewport"content="width=device-width, initial-scale=1"><metaname="color-scheme"content="light dark"><linkrel="stylesheet"href="css/pico.min.css"><title>Hello world!</title></head><body><mainclass="container"><h1>Hello world!</h1></main></body></html>
